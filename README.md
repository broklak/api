#### Getting Started
- install Node js https://nodejs.org/en/

- install dependencies

```shell
$ npm install
```

#### Unit testing
```shell
$ npm run test
```

#### Running

- run dev
```shell
$ npm run dev
```

- run prod
```shell
$ npm start
```
